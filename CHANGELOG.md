# CHANGELOG

Этот файл предназначен для публичного описания крупных изменений проекта. Внутренние номера hotfix можно не переносить дословно, если они раскрывают рабочие детали.

## Основные направления развития

### Тарифы и периоды

- Поддержка тарифов с несколькими сроками, например 30 и 365 дней.
- Отдельные цены для RUB/карта/СБП и Telegram Stars.
- Реферальная скидка по каждому сроку.
- Отдельное название тарифа для страницы покупки.

### Remote nodes

- Управление VPS-node серверами через SSH.
- Генерация sing-box конфигов.
- Поддержка VLESS, Hysteria2 и TUIC.
- Выбор доступных протоколов на node.
- Remote sync и reconciliation.
- Поддержка migration domain preview/apply.

### MultiHop / Double VPN

- Поддержка режимов, в которых трафик может идти цепочкой через основной сервер и второй endpoint.
- Возможные схемы зависят от настройки S-UI/sing-box: main → remote node → internet, main → remote node → WARP, main → WARP endpoint.
- Режим можно не использовать, если проекту достаточно обычных node-подписок.

### Antiabuse

- Учет лимита устройств/IP.
- MAIN + remote node antiabuse.
- Поддержка trial access.
- Дедупликация динамических IP по логическим источникам.
- Предупреждения, счетчики и evidence.
- Ручное отключение подписки.
- Подготовленная логика auto-disable с безопасными ограничениями.

### Manual/offline subscriptions

- Создание подписок администратором для клиентов без Telegram.
- Список, поиск и повторная выдача ссылок.
- Управление сроком, включением, отключением и удалением.
- Telegram binding по TGID или pending binding по username.

### Сертификаты и домен

- Динамические пути сертификатов через `{domain}`.
- S-UI остается источником сертификата.
- Бот мониторит срок действия и копии на node.
- Preview/apply миграции домена node и SNI.

### Monitoring

- Статистика пользователей и воронки.
- Исключение админов из отчетов.
- Production doctor с проверкой сертификатов.
