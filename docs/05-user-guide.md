# Пользовательские функции

## Главное меню

Пользователь обычно видит покупку подписки, продление, мои подписки, платежи, FAQ, профиль, поддержку, Telegram proxy и тестовый доступ. Конкретный набор пунктов зависит от настроек админа.

## Покупка подписки

Пользователь выбирает тариф и видит предпросмотр: срок, стоимость, способы оплаты, локации, лимит устройств/IP и важные условия тарифа.

Если у тарифа есть несколько сроков, кнопки оплаты сразу выбирают срок и способ оплаты.

## Продление

Пользователь выбирает активную подписку и видит доступные варианты продления. Продление использует условия выбранного тарифа и доступные периоды.

## Мои подписки

В разделе отображаются активные подписки пользователя: название тарифа, срок действия, лимит устройств/IP, ссылка подписки, Clash-ссылка, данные Double VPN и текущий счетчик устройств/IP.

Технические слова вроде S-UI, MAIN, API links не должны показываться пользователю.

## Тестовый доступ

Если включен, пользователь может получить тестовую подписку. Условия задаются администратором.

## FAQ и подключение устройства

FAQ может содержать инструкции по приложениям и платформам. Администратор может редактировать FAQ из бота.

## Профиль

В профиле пользователь видит TGID, реферальную ссылку, приглашения и скидку. TGID полезен для привязки manual/offline подписки.

## Поддержка

Пользователь может создать обращение в поддержку. Админ видит тикет и может отвечать из админ-панели.
