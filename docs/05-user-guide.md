# Пользовательские функции

Пользователь может покупать и продлевать подписку, смотреть “Мои подписки”, получать тестовый доступ, читать FAQ, открывать профиль, получать Telegram proxy и писать в поддержку.

В “Мои подписки” отображаются тариф, срок, лимит устройств/IP, ссылка подписки, Clash-ссылка, данные MultiHop / Double VPN если включено, и счетчик устройств/IP.
