# Рассылки

Админская рассылка позволяет отправить сообщение пользователям бота.

## Аудитория

Поддерживаются режимы:

- только обычные подписки;
- активный тестовый доступ;
- обычные подписки + активный тестовый доступ.

Manual/offline подписки попадают в аудиторию только если они привязаны к Telegram.

## Preview

Перед отправкой админ должен увидеть текст сообщения, выбранную аудиторию и примерное количество получателей.

## Исключения

Админы из `admin_ids` не должны попадать в пользовательскую рассылку.

## Ошибки Telegram

Telegram Bot API не сообщает напрямую, что пользователь удалил чат. Но при отправке могут быть ошибки `bot was blocked`, `chat not found`, `user is deactivated`, `forbidden`.

## Ограничения

Рассылки нужно отправлять с учетом rate limits Telegram. Для массовых отправок нужен throttle и обработка `RetryAfter`.
