# Рассылки

Рассылка позволяет администратору отправить сообщение группе пользователей.

## Где находится

```text
🛠 Админ → 📣 Рассылка
```

## Что можно отправить

Рассылка может поддерживать:

```text
текст
картинку
картинку + подпись
URL-кнопку
```

Если отправляется картинка, она должна уйти пользователю как media message.

Если добавлена кнопка, она должна прикрепиться к сообщению.

## Сценарий создания

1. Админ открывает `📣 Рассылка`.
2. Отправляет текст или картинку.
3. Бот сохраняет draft.
4. Бот показывает предпросмотр.
5. Админ выбирает аудиторию.
6. При необходимости добавляет URL-кнопку.
7. Нажимает `✅ Отправить`.

## Аудитория

Режимы:

```text
только обычные подписки
активный тестовый доступ
обычные подписки + активный тестовый доступ
```

## Только обычные подписки

Получатели — пользователи с активной обычной подпиской.

Manual/offline подписки попадут сюда только если они привязаны к Telegram.

## Активный тестовый доступ

Получатели — пользователи с активным trial.

## Обычные + тестовый

Получатели — объединение двух аудиторий.

Если пользователь имеет и trial, и paid, он должен получить сообщение один раз.

## Админы

Админы из `admin_ids` не должны попадать в рассылку.

## URL-кнопка

Формат:

```text
Название | https://example.com
```

## Предпросмотр

Preview должен показывать тип сообщения, текст, наличие картинки, кнопку, выбранную аудиторию и примерное количество получателей.

## Отправка

После подтверждения бот отправляет сообщение получателям.

Важно учитывать Telegram rate limits. Для большой базы нужна задержка между отправками и обработка `RetryAfter`.

## Ошибки отправки

Возможные ошибки:

- bot was blocked;
- chat not found;
- user is deactivated;
- forbidden;
- retry after.

Такие пользователи могут попадать в метрику “недоступны для сообщений”.

## Что проверить

```text
1. Текстовая рассылка.
2. Картинка.
3. Картинка + подпись.
4. URL-кнопка.
5. Только paid.
6. Только trial.
7. Paid + trial.
8. Админы пропускаются.
9. Пользователь не получает дубль.
10. Ошибки Telegram не роняют рассылку.
```
