# Тарифы и периоды

Тариф определяет название, срок, стоимость, лимит устройств/IP, трафик, MultiHop / Double VPN, S-UI inbounds, remote node scope и доступные периоды.

## MultiHop / Double VPN

Опциональный режим multihop-маршрутизации: user → main server → remote server → internet, user → main server → remote server → WARP или user → main server → WARP endpoint. Если режим выключен, пользователю лучше не показывать строку “недоступен”.
