# Карта кнопок и функций

Этот файл нужен, чтобы быстро понять, какая кнопка за что отвечает. Он не заменяет подробные разделы, а служит общей картой проекта.

## Пользовательское меню

| Кнопка | Что делает |
|---|---|
| `🛒 Купить` | Открывает тарифы, предпросмотр и оплату. |
| `🔄 Продлить` | Продлевает существующую подписку по условиям тарифа. |
| `🔑 Мои подписки` | Показывает ссылки, срок, лимит устройств/IP и доступные локации. |
| `💳 Мои платежи` | Показывает историю и статус платежей. |
| `🎁 Тестовый доступ` | Выдает trial, если разрешено настройками. |
| `💎 Telegram proxy` | Показывает Telegram proxy для разрешенной аудитории. |
| `👤 Профиль` | Показывает TGID, username и реферальные данные. |
| `🆘 Поддержка` | Создает тикет поддержки. |
| `❓ FAQ` | Открывает пользовательскую справку. |
| `ℹ️ Информация` | Открывает информационный раздел. |

## Remote nodes: важные кнопки

| Кнопка | Что делает |
|---|---|
| `🖥 Управление VPS-локациями` | Список node из `nodes.json`. |
| `➕ Добавить VPS-локацию` | Добавляет новую node в конфиг. |
| `🚀 Подготовить VPS (bootstrap)` | Подключается по SSH и готовит node: sing-box, dirs, certs, config, firewall, service. |
| `🔍 Проверить эту VPS` | Проверяет доступность node, sing-box, config и certificates. |
| `🔄 Синхронизировать эту локацию` | Пересобирает config конкретной node и применяет его. |
| `🚰 Убрать локацию из подписок` | Включает drain: node убирается из выдачи links. |
| `🧹 Убрать ссылки этой локации` | Удаляет managed links выбранной node из подписок. |
| `🔐 Перегенерировать Reality/HY2 ключи` | Меняет ключи/secret протоколов node. |
| `⚫ Отключить VPS-локацию` | Выключает node. |
| `🟡 Включить обслуживание` | Переводит node в maintenance. |
| `✏️ Название страны/локации` | Меняет display name. |
| `🧩 JSON-редактор VPS` | Ручное редактирование JSON node. |
| `💎 Управлять TG PROXY` | Управление mtg proxy на node. |
| `🔍 Предпросмотр миграции домена` | Показывает, что изменится при смене домена. |
| `✅ Применить миграцию домена` | Делает backup, обновляет managed domains/SNI/cert paths и ставит sync. |

## Manual/offline подписки: важные кнопки

| Кнопка | Что делает |
|---|---|
| `➕ Создать manual/offline подписку` | Создает подписку вручную по тарифу и сроку. |
| `🔎 Поиск` | Ищет manual-подписку. |
| `🔗 Показать ссылки` | Показывает subscription link и Clash link. |
| `✏️ Изменить данные` | Меняет имя, контакт и заметку. |
| `🔗 Telegram` | Привязывает TGID, TGID+username или pending username. |
| `🧩 Double VPN` | Включает/выключает MultiHop override для этой подписки. |
| `🔄 Продлить` | Продлевает manual-подписку по тарифу. |
| `⛔ Отключить / ✅ Включить` | Отключает или включает подписку. |
| `🗑 Удалить` | Удаляет подписку и managed remote data. |

## Antiabuse: важные кнопки

| Кнопка | Что делает |
|---|---|
| `🟡 На проверке` | Помечает кейс как проверяемый. |
| `✅ Норма` | Помечает кейс как нормальный. |
| `🙈 Игнор` | Игнорирует кейс. |
| `📝 Заметка` | Добавляет заметку. |
| `✏️ Лимит IP` | Задает override лимита. |
| `♻️ Сбросить лимит` | Возвращает лимит тарифа/теста. |
| `⛔ Отключить на 10 мин` | Вручную отключает подписку на 10 минут. |
| `⏱ Свой срок` | Отключает на заданный срок. |
| `✅ Включить сейчас` | Снимает ограничение. |
| `📄 Auto-disable evidence` | Показывает доказательную базу. |
| `⚙️ Auto-disable` | Настройки автоматического отключения. |

## Рассылка: важные кнопки

| Кнопка | Что делает |
|---|---|
| `🎯 Кому` | Показывает выбранную аудиторию. |
| `Обычные` | Только активные обычные подписки. |
| `Тест` | Только активный тестовый доступ. |
| `Обычные + тест` | Обе аудитории. |
| `🔗 Добавить кнопку` | Добавляет URL-кнопку к сообщению. |
| `🗑 Убрать кнопку` | Убирает URL-кнопку. |
| `✅ Отправить` | Запускает рассылку. |

## Production doctor: важные кнопки

| Кнопка | Что делает |
|---|---|
| `🔄 Обновить` | Повторяет проверку. |
| `🔁 Сверка локаций и ссылок` | Проверяет соответствие S-UI, local state, node configs и links. |
| `🔗 Восстановить ссылки локаций` | Принудительно восстанавливает managed remote links. |
| `🧹 Очистка хвостов` | Чистит stale/ghost managed data. |
| `🔄 Запустить sync локаций` | Ставит remote sync/deploy. |
| `💳 Аудит зависших оплат` | Ищет pending/apply_failed платежи. |
| `🧬 Миграция server_code` | Помогает сменить server_code. |
