# Antiabuse

Antiabuse отслеживает превышение лимита устройств/IP. Бот считает активные уникальные IP или логические источники IP, а не физические устройства.

Поддерживаются MAIN journal parsing, remote node collector, dynamic IP dedupe, warning counters, ручное отключение, evidence и безопасная модель auto-disable.
