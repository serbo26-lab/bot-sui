# Remote nodes

Remote node — отдельный VPS с sing-box. Бот копирует сертификаты, генерирует config, деплоит node, перезапускает sing-box и обновляет external links.

Поддерживаются VLESS/Reality, Hysteria2, TUIC, protocol selection, sync/deploy, reconciliation, drain/maintenance и domain migration preview/apply.
