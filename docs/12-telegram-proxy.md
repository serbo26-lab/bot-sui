# Telegram proxy

Telegram proxy помогает пользователю подключаться к Telegram. Это отдельная функция и не заменяет VPN-подписку.

## Где устанавливается

Proxy может устанавливаться на remote node. Обычно используется `mtg`.

## Управление

Админ может установить proxy, проверить состояние, переустановить, перегенерировать secret и удалить proxy.

Повторная установка не должна молча перезаписывать рабочий proxy. Лучше показать экран управления.

## Domain/SNI

Для proxy лучше использовать `node.domain` или явно указанный домен/SNI. IP не должен быть default-вариантом для SNI.

## Версия mtg

Версию лучше фиксировать, например `v2.2.8`, а не использовать latest без контроля.

## Доступ пользователей

В админке можно выбрать режим: только обычные подписки или обычные подписки + активный тестовый доступ.

## Записи в базе

Auto-added proxy records должны обновлять существующие managed records, а не создавать дубли.
