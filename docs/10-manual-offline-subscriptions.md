# Manual/offline подписки

Manual/offline подписки нужны для клиентов без Telegram. Подписка создается админом, привязана к тарифу, сроку, лимитам, remote nodes и antiabuse.

Поддерживается MANUAL_ID, редактирование имени/контакта/заметки, MultiHop / Double VPN override, привязка к Telegram по TGID или pending binding по username. После привязки подписка появляется в “Мои подписки”.
