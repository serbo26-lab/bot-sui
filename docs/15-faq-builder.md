# FAQ-конструктор

Администратор может редактировать пользовательский FAQ без изменения кода.

## Возможности

- редактировать главный текст FAQ;
- редактировать разделы;
- добавлять разделы и подразделы;
- добавлять URL-кнопки;
- менять название кнопки, текст и URL;
- включать или скрывать разделы;
- двигать раздел выше/ниже;
- удалять разделы;
- смотреть preview;
- сбрасывать FAQ на стандартный.

## URL-кнопки

Формат:

```text
Название | https://example.com
```

## Защита от перезаписи

Пользовательский FAQ не должен перезаписываться при обновлении билда. Стандартный FAQ импортируется только один раз. Если в базе уже есть пользовательский FAQ, обновление билда его не трогает. Сброс выполняется только по явной кнопке админа.

## Callback data

Админу не нужно вручную вводить callback data. Это внутренняя техническая часть.
