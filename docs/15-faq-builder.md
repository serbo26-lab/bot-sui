# FAQ-конструктор

FAQ-конструктор позволяет редактировать пользовательский FAQ из админки без изменения кода.

## Где находится

```text
🛠 Админ → ❓ Пользовательский FAQ
```

## Что видит пользователь

Пользователь нажимает:

```text
❓ FAQ
```

и видит главный текст FAQ, список разделов, подразделы и URL-кнопки, если они добавлены.

## Главный текст FAQ

Это текст верхнего экрана FAQ.

Админ может изменить его, например:

```text
Выберите раздел с инструкцией ниже.
```

## Разделы

Раздел — это кнопка внутри FAQ.

Примеры:

```text
📱 Подключить устройство
💳 Оплата
⭐ Telegram Stars
🛡 Лимит устройств/IP
💎 Telegram proxy
```

## Подразделы

Раздел может иметь подразделы.

Пример:

```text
📱 Подключить устройство
├─ iPhone / iPad
├─ Android
├─ Windows
└─ Router
```

## URL-кнопки

Формат ввода:

```text
Название | https://example.com
```

## Включить / скрыть

Раздел можно скрыть, не удаляя его.

Это полезно, если инструкция еще не готова.

## Вверх / вниз

Разделы можно двигать выше или ниже, чтобы менять порядок кнопок.

## Удалить

Удаляет раздел. Перед удалением лучше использовать confirm.

## Предпросмотр

Preview показывает, как FAQ будет выглядеть для пользователя.

## Сброс на стандартный FAQ

Админ может сбросить FAQ на стандартный.

Важно: обновление билда не должно само перезаписывать кастомный FAQ.

Правильная логика:

```text
первый запуск → стандартный FAQ импортируется
админ изменил FAQ → хранится в базе
обновление билда → FAQ не трогается
админ нажал reset → стандартный FAQ импортируется снова
```

## Что не нужно показывать админу

Админу не нужно вручную вводить callback data или технические ID.

## Что проверить

```text
1. Главный текст меняется.
2. Раздел добавляется.
3. Подраздел добавляется.
4. URL-кнопка работает.
5. Раздел скрывается.
6. Раздел двигается.
7. Preview работает.
8. Reset работает.
9. Обновление билда не перезаписывает FAQ.
```
