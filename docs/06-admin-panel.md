# Админ-панель

Админ-панель открывается администраторам из `admin_ids`.

Главный экран должен оставаться коротким. Подробные объяснения лучше хранить в FAQ по админке и документации.

## Основные разделы

- Тарифы.
- Платежи.
- Manual/offline подписки.
- Remote nodes / локации.
- Antiabuse.
- Telegram proxy.
- Рассылка.
- Мониторинг.
- Production doctor.
- Пользовательский FAQ.
- Поддержка.

## Тарифы
Админ создает и редактирует тарифы, сроки, цены, лимиты и доступные node.

## Платежи
Админ видит платежи, статусы, apply_failed и может выполнить approve/retry, если это предусмотрено кодом.

## Manual/offline подписки
Админ может создать подписку без Telegram-пользователя, а затем привязать ее к TGID или username.

## Remote nodes
Раздел управляет node-серверами, протоколами, sync/deploy, domain migration и Telegram proxy.

## Antiabuse
Показывает кейсы превышения лимита устройств/IP, предупреждения, evidence и ручные ограничения.

## Production doctor
Показывает состояние ключевых компонентов: S-UI, база, remote nodes, certificates, antiabuse и другие проверки.
