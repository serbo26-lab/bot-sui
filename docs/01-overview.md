# Обзор проекта

Your VPN Bot — Telegram-бот для управления VPN-подписками на базе S-UI и sing-box.

Проект находится в активной разработке и предназначен для самостоятельной адаптации под свою инфраструктуру.

## Назначение

Бот закрывает несколько задач: пользователь покупает или продлевает подписку; администратор управляет тарифами, платежами и пользователями; remote node-серверы добавляются как VPN-локации; antiabuse отслеживает превышение лимита устройств/IP; manual/offline режим позволяет выдавать подписки клиентам без Telegram.

## Архитектура

```text
Telegram
↓
aiogram bot
↓
S-UI API
↓
main sing-box
↓
remote node sing-box
```

Бот не заменяет S-UI. S-UI остается важной частью системы: через API бот создает и обновляет клиентов, а также использует подписочные ссылки.

Remote node-локации управляются ботом отдельно. Бот генерирует sing-box config для node, копирует сертификаты, обновляет external links и выполняет reconciliation.

## Что проект не делает

- Не устанавливает S-UI без подготовки сервера.
- Не является готовым коммерческим SaaS-продуктом.
- Не освобождает администратора от настройки домена, DNS, сертификатов, firewall и systemd.
- Не должен публиковаться с реальными секретами.
